# AI Manju Automatic Production System

English version of the project documentation.

## Overview

An end-to-end AI-powered automatic comic drama production system that transforms story outlines into complete animated videos with characters, storyboards, voice acting, and music.

## Key Features

- 🤖 AI Script Generation
- 🎨 Character Consistency (IPAdapter + LoRA)
- 🎬 Automatic Storyboard Generation
- 🎥 Image-to-Video Animation
- 🎵 AI Voice & Music Synthesis
- ⚡ Fully Automated Pipeline

## Tech Stack

- **LLM**: Claude 3.5 / GPT-4
- **Image**: Stable Diffusion XL + ComfyUI
- **Video**: Kling AI / Dreamina / AnimateDiff
- **Voice**: ElevenLabs / Azure TTS
- **Music**: Suno AI / Udio
- **Backend**: FastAPI + Celery + Redis

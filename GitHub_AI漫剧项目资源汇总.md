## 🎨 角色一致性项目

### 🔥 必装工具

#### 1. ComfyUI_IPAdapter_plus ⭐⭐⭐⭐⭐
- **链接**: https://github.com/cubiq/ComfyUI_IPAdapter_plus
- **功能**: IPAdapter ComfyUI插件，图像提示适配
- **用途**: 角色面部一致性保持
- **安装**: ComfyUI自定义节点

#### 2. StoryDiffusion ⭐⭐⭐⭐⭐
- **链接**: https://github.com/HVision-NKU/StoryDiffusion
- **功能**: 官方角色一致性漫画生成
- **用途**: 长故事多角色保持一致性
- **特点**: 自注意力机制保持角色统一

#### 3. ComfyUI-ReActor ⭐⭐⭐⭐
- **链接**: https://github.com/Gourieff/ComfyUI-ReActor
- **功能**: 快速替换生成图像中的人脸
- **用途**: 角色换脸、一致性修复
- **特点**: 速度快，效果好

#### 4. ComfyUI-PuLID-Flux-Enhanced ⭐⭐⭐⭐
- **链接**: https://github.com/sipie800/ComfyUI-PuLID-Flux-Enhanced
- **功能**: FLUX模型的角色一致性
- **用途**: FLUX模型角色保持
- **特点**: 支持FLUX最新模型

#### 5. PhotoMaker ⭐⭐⭐⭐
- **链接**: https://github.com/TencentARC/PhotoMaker
- **功能**: 堆叠ID嵌入保持人物特征
- **用途**: 真实人物照片风格迁移
- **特点**: 腾讯开源，质量高

#### 6. ComfyUI-layerdiffuse ⭐⭐⭐⭐
- **链接**: https://github.com/huchenlei/ComfyUI-layerdiffuse
- **功能**: 生成透明图层
- **用途**: 漫画分层、背景分离
- **特点**: 支持透明PNG生成

#### 7. InstantID ⭐⭐⭐⭐
- **链接**: https://github.com/InstantID/InstantID
- **功能**: 零样本身份保持
- **用途**: 单图角色保持
- **特点**: 无需训练，即插即用

#### 8. ComfyUI-Prompt-Schedule ⭐⭐⭐
- **链接**: https://github.com/DrMWeigand/ComfyUI-Prompt-Schedule
- **功能**: 提示词时间轴控制
- **用途**: 批量生成时变化控制

#### 9. ComfyUI-VideoHelperSuite ⭐⭐⭐

## 📚 推荐技术组合

### 方案一：经典方案（SD1.5/SDXL）
```
角色一致性: IPAdapter + StoryDiffusion + ReActor
分镜控制: ControlNet (OpenPose/Canny/Depth)
视频生成: AnimateDiff
配音: GPT-SoVITS + ChatTTS
音乐: YuE
工作流: ComfyUI + Manager + Workspace
```

### 方案二：FLUX方案（推荐）
```
基础模型: FLUX
角色一致性: PuLID-Flux + IPAdapter-Flux
分层处理: LayerDiffuse
视频生成: 可灵/即梦 API + AnimateDiff
配音: GPT-SoVITS
音乐: YuE/Suno
```

### 方案三：完整工作流
```
分镜: PanelForge
角色: StoryDiffusion + InstantID
分层: LayerDiffuse
视频: AnimateDiff-Evolved
后期: Impact-Pack + VideoHelperSuite
管理: Workspace Manager
```

## 🔧 安装命令速查

### ComfyUI 插件安装
```bash
# 进入ComfyUI自定义节点目录
cd ComfyUI/custom_nodes

# 安装Manager（必须先装）
git clone https://github.com/ltdrdata/ComfyUI-Manager.git

# 安装IPAdapter
git clone https://github.com/cubiq/ComfyUI_IPAdapter_plus.git

# 安装StoryDiffusion
git clone https://github.com/HVision-NKU/StoryDiffusion.git

# 安装ReActor
git clone https://github.com/Gourieff/ComfyUI-ReActor.git

# 安装LayerDiffuse
git clone https://github.com/huchenlei/ComfyUI-layerdiffuse.git

# 安装Impact-Pack
git clone https://github.com/ltdrdata/ComfyUI-Impact-Pack.git

# 重启ComfyUI
```

### Python 环境安装
```bash
# GPT-SoVITS
git clone https://github.com/RVC-Boss/GPT-SoVITS.git
cd GPT-SoVITS
pip install -r requirements.txt

# ChatTTS
git clone https://github.com/2noise/ChatTTS.git
cd ChatTTS
pip install -r requirements.txt

# YuE
git clone https://github.com/yu-ai/YuE.git
cd YuE
pip install -r requirements.txt
```

## ⚠️ 注意事项

1. **硬件要求**: 多数项目需要NVIDIA GPU，建议12GB+显存
2. **模型版权**: 注意商用模型的授权协议
